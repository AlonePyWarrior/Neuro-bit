Basic — ۲M/ماه

  • بدون کلاس آنلاین فنی؛ ارتباط با منتور به‌صورت چت + بازخورد.
  • جلسهٔ هفتگی «متا» (توسعه فردی/ذهنیت/نظم).
  • Step-based: عبور هر قدم فقط با تأیید تسک؛ هر دوره تعداد قدم‌های مشخص خود را دارد.

Plus — ۴M/ماه

  • همهٔ امکانات Basic + جلسات گروهی فنی هفتگی (آنلاین، حداکثر ۱۰ نفر).
  • حفظ چت منتور + متا + ارزیابی قدم‌به‌قدم.

Pro — ۸M/ماه

  • مسیر ورزشی شخصی‌سازی‌شده (تمرین/تغذیه با پایش).
  • جلسات خصوصی فنی ۶۰ دقیقه‌ای — هفتگی (۱:۱).
  • اولویت مسیر کارآموزی.

Ultimate — ۱۶M/ماه (پیشنهادی)

  • منتور ارشد + بوت‌کمپ فشرده + پرتفولیو کامل.
  • اولویت ویژه در معرفی کارآموزی.

A) شراکت کانالی (Channel)

  • یاسان: کانال/لید، PR، دسترسی مدارس
  • نوروبیت: محصول، اجرا، منتورها
  • Revenue Share پایلوت: ۲۰٪ روی ثبت‌نام‌های منشأ یاسان
  • SLA: ≥۱۲۰ لید واجدشرط/۹۰ روز؛ TTR<48h
  • انحصار محدود: فقط روی کانال‌های یاسان و شهرهای توافقی (≤۱۲ ماه، مشروط به SLA)

B) واحد رشد درون یاسان (Spin-in)

  • واحد «نوروبیت» با P&L مستقل
  • کمیته راهبری مشترک؛ Board Observer برای یاسان
  • حق وتو محدود (برند/قانون)

C) JV سبک (بعد از پایلوت)

  • پس از اثبات (مثلاً ۳۰۰+ دانش‌آموز/سالیانه)
  • مذاکره سهم‌ها + حق خروج + بازنگری ارزش‌گذاری

دامنه و بودجه

  • دامنه: یک خوشهٔ مدرسه + جامعهٔ فارغ‌التحصیلان یاسان

  • زیرساخت (نوروبیت): ۰.۵–۱.۵M/ماه
حاکمیت: چک‌این هفتگی، کمیته راهبری ماهانه، مرور فصلی
